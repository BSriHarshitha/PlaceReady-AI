## Overview

PlaceReady AI is a comprehensive placement preparation platform that leverages Google technologies to analyze student readiness and provide personalized learning recommendations.

## Features

- **AI Resume Analysis** - Google Cloud AI powered resume parsing
- **Skill Assessment** - Comprehensive skill gap analysis
- **Personalized Learning** - Tailored learning paths and recommendations
- **Admin Dashboard** - Real-time analytics and student progress tracking
- **Google Technologies Integration** - 7+ Google services integrated

## Google Technologies Used

1. **Firebase Authentication** - Secure user management
2. **Firebase Hosting** - Scalable web deployment
3. **Google Cloud AI** - Intelligent resume parsing
4. **Google Analytics** - User behavior tracking
5. **Google Drive API** - Document storage
6. **Google Calendar** - Interview scheduling
7. **Google Meet** - Virtual interviews

## Tech Stack

- **Frontend**: React 19, TypeScript, Material-UI

- **AI/ML**: Google Cloud AI, TensorFlow.js
- **Analytics**: Google Analytics
- **Deployment**: GitHub Pages

## Quick Start

1. Clone the repository
2. Install dependencies: `cd frontend && npm install`
3. Start development server: `npm start`
4. Access at `http://localhost:3000`

## Deployment

Deploy to GitHub Pages:
```bash
cd frontend
npm run deploy
```
